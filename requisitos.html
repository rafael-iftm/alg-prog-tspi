Resumo dos Requisitos Mínimos do Trabalho

Classes:

  • Criar no mínimo 3 classes: uma classe principal (Programa) e duas outras classes, sendo que uma fará agregação por composição na outra.
  • Implementar no mínimo 3 atributos para a classe principal e 2 para a classe secundária, sendo um deles um HashMap.
  • Utilizar ArrayList para implementar a composição, ou seja, a classe principal terá uma lista contendo objetos da classe secundária.
  • Implementar construtores, métodos getters e setters, e outros métodos necessários.

Classe Programa:


public class Programa {
    private static Consulta consulta = new Consulta();
    private static ArrayList<Remedio> remediosCadastrados = new ArrayList<>();
    private static HashMap<String, String> unidades = new HashMap<>();
}

Menu:

  • Apresentar um menu na classe Programa com pelo menos 10 opções de funções para o usuário.
  • O menu deve permitir manipular o objeto da classe principal, objetos da classe secundária e o dicionário implementado pelo HashMap.

Implementação de Operações:

  • Implementar métodos estáticos na classe Programa para cada opção do menu (mínimo de 10 operações).
  • Exemplos de operações: cadastrar, editar, apagar, consultar, listar elementos, etc.

Validações:

  • Fazer validações dos dados de entrada, criando regras próprias para garantir a integridade dos dados.
